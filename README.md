# Admin Dashboard

A simple dashboard layout built using HTML and CSS Grid.

## Features
- Sidebar navigation
- Header with search bar and buttons
- Project cards layout
- Announcements and trending section
- Organized assets folder (icons, images, fonts)

## Project Structure

│── index.html

│── styles.css

└── assets/

├── icons/

└── images/


## How to Use
1. Download or clone the repository
2. Open `index.html` in your browser

## Built With
- HTML5
- CSS3 (Grid & Flexbox)
